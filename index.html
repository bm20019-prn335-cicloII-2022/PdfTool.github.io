PdfTool 1.0

Basado en itextSharp 5

Pdftool es una herramienta para la manipulacion de pdf
escrita en CSharp (C#), utilizando el .NET6.

  1. Convierte Imagenes en pdf.
  2. Fusiona Pdfs
  3. Combina pdfs con imagenes
  4. Agregar o Modifica Atributos

    1. Autor
    2. Creador
    3. Sujeto
    4. Language

Ejemplos de uso

PdfTool -i "img1.png" "img2.png" "img3.png" -o resultImagePdf.pdf
PdfTool -i "img1.png" "img2.png" "img3.png"
PdfTool -i "FilePdf1.png" "FilePdf2.png" "FilePdf3.png" -o resultMerge.pdf
PdfTool -i "img1.png" "FilePdf2.png" "img3.png" -o resultCombine.pdf
